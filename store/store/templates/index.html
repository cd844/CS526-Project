Main Project Goal

Github has become the most popular social coding platform.
As such, it is an interesting issue to analyze the social factors of Github users and repositories.
With Software engineering field evolving at a faster rate, the applicants are expected to be competitve.
To stand out, it is important for applicants to demonstrate their skills. Github comes to your rescue.

Our Application analyzes the available repositories and advices the user what projects are in th trend,
followed by the languages they are built with. This gives a sense of direction to the user!

Dataset Description

This dataset has been scrapped by us using web crawling tools and only the important attributes have been taken into consideration .



Set of Questions to be Answered

  • What kind of projects are viewed/forked the most?
  • What are the common languages?
  • What languages to be used for the kind of projects you mentioned
  • Time series of repo creation

Programming Languages, Libraries, and Tools

  • Use of Plotly to create interactive plots
  • Connect the sqlite database with the flask for querying
  • Use of K means algorithm for clustering
  • Use of Pyspark
  • Project to be integrated on Flask
  • Use of GraphX for visualization
